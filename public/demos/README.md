# 🏊‍♂️ Club de Plongée Saint-Leu

Site vitrine premium pour club de plongée à Saint-Leu, La Réunion.

## 🚀 Lancement Rapide

```bash
# Aller dans le dossier
cd club-plongee-saint-leu

# Lancer le serveur
python3 -m http.server 8080

# Accéder au site
# http://localhost:8080
```
